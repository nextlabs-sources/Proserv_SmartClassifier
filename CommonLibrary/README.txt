Installation steps :

1. Unzip the Watcher-xxxx.zip to C:\

2. The directory stucture should be as below :

	C:\
	----Watcher\
						----conf\
						----setup\
						----logs\
						----xlib\
						----nextlabs-scanner.jar
						----README.txt(this file)
						   
3. You will need to set the JAVA_HOME to Java 7 64 bit folder.

4. Run the start.bat from C:\Watcher\setup folder, this will start the scanner
