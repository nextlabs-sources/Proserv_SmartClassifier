Installation steps :

1. Unzip the Watcher-xxxx.zip to C:\

2. The directory structure should be as below :

	C:\
	----CASExtension\
						----conf\
						----setup\
						----logs\
						----xlib\
						----nextlabs-cas-extension.jar
						----README.txt(this file)
						   
3. You will need to set the JAVA_HOME to Java 7 64 bit folder.
